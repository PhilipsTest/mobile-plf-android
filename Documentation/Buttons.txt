The theme colour will be applied to all the buttons by default. No other styling is nessecary.

For outlined buttons add style attribute 
style="?attr/outlinedButtonStyle"

For outlined transparent buttons add style attribute
style="?attr/outlinedButtonTransparentStyle"

For white transparent buttons add style attribute
style="@style/Button.TransparentWhite"


If you want buttons which do not belong to the theme, apply style in your layout XML.
style="@style/Button.Solid.BrightAqua"
style="@style/Button.OutlinedBrightAqua"
style="@style/Button.TransparentBrightAqua"
